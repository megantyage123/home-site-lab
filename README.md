# home-site-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面，这些页面彼此之间没有强关联，也不属于某个特定网站或服务。每个页面都可单独访问，适合作为个人实验、静态展示或临时页面的存档区域。

## 目录说明

```
home-site-lab/
├── README.md          # 本文件
├── pages/             # 存放各个 HTML 页面的目录
│   ├── example-page/  # 示例页面文件夹（可自行创建）
│   └── ...
└── assets/            # 公共资源文件（如 CSS、JS、图片等）
```

- `pages/`：每个独立页面建议放在自己的子目录中，便于管理。
- `assets/`：存放多个页面共用的资源，避免重复引用。

## 页面归档说明

- 每个页面以独立子目录的形式组织，目录名即页面标识。
- 页面内容无特定限制，但应保持基本可读性和功能完整性。
- 不要求页面之间风格统一，允许各自独立设计。
- 所有页面均通过 GitHub Pages 或其他静态托管服务发布，访问路径与目录结构一致。

## 维护说明

- 欢迎提交新的 HTML 页面，请放置在 `pages/` 下并保持目录结构清晰。
- 如需修改已有页面，请直接更新对应文件。
- 本仓库不保证所有页面长期可用，老旧页面可能被归档或移除。
- 提交前请确保页面不包含敏感信息或外部依赖失效。
